# 터미널 없이 실행 관리

종목 검색 서버를 터미널 창 없이 백그라운드에서 켜기 위한 파일들입니다.

## 한 번만 등록

`터미널없이_자동실행_등록.command`를 더블클릭하세요.

등록 후에는 브라우저에서 아래 주소만 열면 됩니다.

```text
http://127.0.0.1:8765
```

## 자동실행 해제

`터미널없이_자동실행_해제.command`를 더블클릭하세요.

## 참고

- 최초 1회는 프로젝트 루트의 `종목검색실행.command`로 검색 인덱스를 만들어야 합니다.
- 최신 유튜브 자막으로 갱신하고 싶을 때도 `종목검색실행.command`를 실행하세요.
- 이 폴더의 `*.sh`와 `*.plist`는 등록/해제 커맨드가 내부적으로 쓰는 파일입니다.
